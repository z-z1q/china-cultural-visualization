## 数据来源

本项目数据主要来源于互联网。

主要数据类型：

- **民居样本数据**（residences.csv）：互联网，包含50个中国传统民居样本
- **地图边界数据**（china_full.geojson）：来源于自然资源部标准地图服务，审图号 GS(2024)0654 号

## 数据处理说明

### 防震评分计算

防震评分通过多因素加权计算得出：

```
防震评分 = 100 × (
    0.50 × 墙体厚度归一化值 +
    0.18 × (1 - 开窗比例) +
    0.12 × (1 - 面积归一化值) +
    0.12 × (1 - 屋顶坡度归一化值) +
    0.08 × 材料评分
)
```

#### 材料评分规则

| 材料类型     | 评分调整      |
| ------------ | ------------- |
| 木材         | +0.12         |
| 砖石/混凝土  | +0.05 ~ +0.10 |
| 竹/毛毡/茅草 | +0.04         |
| 石材         | -0.06         |
| 土/夯土      | -0.04         |

#### 防震等级划分

| 评分范围 | 等级 |
| -------- | ---- |
| 0-35     | Ⅰ    |
| 35-50    | Ⅱ    |
| 50-65    | Ⅲ    |
| 65-80    | Ⅳ    |
| 80-100   | Ⅴ    |

### 数据归一化

使用 Min-Max 归一化将数据映射到 [0, 1] 区间：

```
归一化值 = (原始值 - 最小值) / (最大值 - 最小值)
```

## 注意事项

### 数据质量声明

1. **数据来源**：本项目民居数据由团队成员收集整理，主要用于计算机设计大赛数据可视化展示
2. **估算数据**：部分建筑参数（如墙体厚度、开窗比例等）为基于文献资料和行业经验的合理估算
3. **简化模型**：防震评分为简化计算模型，仅供可视化展示参考，非专业抗震评估
4. **气候数据**：气候指标为该地区典型值，非特定建筑的实测数据

### 使用建议

1. 本项目数据主要用于教育和展示目的
2. 如需用于专业研究，请查阅权威建筑数据库和文献资料
3. 欢迎对数据提出修正建议
